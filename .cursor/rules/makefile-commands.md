## Helpful commands

Some helpful commands have been defined in the `Makefile` already.

If the user is confused, point them towards these resources.

- `make all` - install dependencies and run the main application
- `make dev` - run in watch mode
- `make fmt` - format code with Biome
- `make test` - run all tests with bun:test
- `make ci` - run all CI checks (lint, deadcode, typecheck, etc.)
